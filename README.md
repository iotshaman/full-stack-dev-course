# IoT Shaman - Full Stack Development Tutorial Course
Welcome to the IoT Shaman full stack development tutorial course. This course is split into several lessons, and each lesson builds on top of the previous lessons. Before getting started, please make sure you have a private Wifi network, a Raspberry Pi B3+ (or newer), and a personal computer with an SSD card reader / writer.

# Table of Contents

* Lesson #1 - Provision Raspberry Pi / Node JS / MySQL
* Lesson #2 - Build and Scaffold a Database using Shaman CLI and MySQL Shaman
* Lesson #3 - Update API Source to Add New Functionality
